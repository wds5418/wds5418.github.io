善用工具

工欲善其事，必先利其器。好的工具一程度上能够压缩工作量，来使我们更高效的挖掘flag

dirsearch（目录扫描工具）

常用命令：

python dirsearch.py -u URL -e php

-u 指定目标ip

-e 指定网站语言

-w 加上自己的字典

-random-agents使用代理

sqlmap（针对sql注入的工具）

常用命令：

sqlmap -u “URL” 指定目标

sqlmap -m 1.txt 获取多个url

sqlmap -r URL.txt 从文件中加载http请求

sqlmap -g 利用谷歌获取目标

sqlmap -l burp.txt

—dbs 查看所有可用数据库

-D<库名> –tables（查看某库下面的可用表）

-D<库名> -T<对应的数据库表名> – column（查询某数据库某表下的所有列名）
